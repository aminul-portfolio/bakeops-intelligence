How to Generate the Export Pack

Run the BI export command after seed and metric build.

The export pack is generated into the project {{ output_folder }} folder. Recommended local sequence:

1. Seed demo data python manage.py seed_demo_data --reset
2. Build metrics python manage.py build_bakery_metrics
3. Export CSV pack {{ export_command }}
4. Review files {{ output_folder }}
